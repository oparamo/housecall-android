Texas Instruments Inc.SensorTag Andorid appRelease Notes--------------------------------------------------------------------------------------------------------------------------------------------------------------Version 1.1.0September 17th, 2013This installer contains the SensorTag Android application package file (.apk). When installed on an Bluetooth 4.0 compatible Android device running Android version 4.3 or newer this app can be used to communicate with the SensorTag.Installation guide1. Copy the SensorTag.apk file to your Android device2. Enable installation of apps from unknown sources (Settings -> Security -> Device Administration -> Unknown Sources)3. Open file manager and launch, locate the .apk file and install the app by clicking the .apk file.For technical support and more information, please see the SensorTag product page (www.ti.com/sensortag), the SensorTag wiki page (www.ti.com/sensortag-wiki) or visit Texas Instruments Bluetooth low energy E2E Forum:http://e2e.ti.com/support/low_power_rf/f/538.aspx--------------------------------------------------------------------------------------------------------------------------------------------------------------
